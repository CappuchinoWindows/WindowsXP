Recreación nostálgica de la interfaz de Windows XP utilizando tecnologías web modernas. Este proyecto simula el icónico sistema operativo con su característico escritorio, menú de inicio, barra de tareas y ventanas interactivas.

Características principales:
- Interfaz fiel al diseño original de Windows XP
- Ventanas arrastrables y redimensionables
- Menú de inicio funcional con aplicaciones simuladas
- Barra de tareas con reloj y área de notificaciones
- Aplicaciones clásicas como Buscaminas, Paint y Bloc de notas
- Efectos de sonido y animaciones auténticas
- Diseño totalmente responsive

Tecnologías: HTML5, CSS3 y JavaScript vanilla 

Este proyecto es puramente educativo y nostálgico, sin afiliación con Microsoft.
